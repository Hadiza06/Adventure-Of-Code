# Advent of Code 2025

Advent of Code https://adventofcode.com/ propose chaque jour en décembre un petit puzzle de programmation. Chaque défi permet de collecter une étoile en résolvant le problème avec du code.

## Contenu
Chaque dossier dayX contient :
- le code pour résoudre le puzzle,
- le fichier d'entrée pour ce jour,
- éventuellement un petit README avec les consignes et la méthode utilisée.
